# 마우스 자동 클릭 애플리케이션 (Python & Chrome Extension)

이 프로젝트는 Python 기반의 데스크톱 앱과 브라우저 전용 Chrome Extension 두 가지 버전의 마우스 자동화 도구를 제공합니다.

---

## 1. Chrome Extension 버전 (MouseAuto Sidebar)

브라우저 우측 사이드바(Side Panel)에서 동작하는 현대적인 마우스 자동화 도구입니다.

### 주요 특징
- **다중 탭 백그라운드 지원**: 여러 탭을 동시에 자동화하며, 탭이 활성화되어 있지 않아도 동작합니다.
- **중앙 제어 타이머**: 익스텐션 사이드바에서 타이밍을 관리하여 브라우저의 전력 절약 기능 방해를 받지 않습니다.
- **프리미엄 UI**: 글래스모피즘이 적용된 다크 모드 인터페이스를 제공합니다.

### 사용법
1. `chrome://extensions` 접속 후 **개발자 모드** 활성화
2. `extension` 폴더를 **압축해제된 확장 프로그램으로 로드**
3. 익스텐션 아이콘을 눌러 사이드바를 열고, 자동화를 원하는 탭에서 **[현재 탭 추가]** 클릭
4. **[녹화 시작]**으로 동작을 기록하고 **[자동 실행]**으로 반복 실행

---

## 2. Python 데스크톱 버전

Python으로 만든 마우스 자동 클릭 애플리케이션입니다. 시스템 전체 마우스 동작을 녹화하고 자동으로 재생할 수 있습니다.

### 주요 기능
- **마우스 녹화**: 마우스 클릭 동작을 실시간으로 녹화
- **자동 재생**: 녹화된 동작을 정확하게 재생
- **자동 실행**: 설정한 시간 간격으로 자동 반복 실행
- **전역 핫키**: F1, F2, ESC 키로 편리한 제어

### 설치 및 실행
```bash
pip install -e .
python main.py
```
