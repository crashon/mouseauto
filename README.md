# 마우스 자동 클릭 애플리케이션

Python으로 만든 마우스 자동 클릭 애플리케이션입니다. 마우스 동작을 녹화하고 자동으로 재생할 수 있습니다.

## 주요 기능

- **마우스 녹화**: 마우스 클릭 동작을 실시간으로 녹화
- **자동 재생**: 녹화된 동작을 정확하게 재생
- **자동 실행**: 설정한 시간 간격으로 자동 반복 실행
- **파일 저장/로드**: 녹화된 동작을 JSON 파일로 저장하고 불러오기
- **전역 핫키**: F1, F2, ESC 키로 편리한 제어
- **직관적인 GUI**: tkinter 기반의 사용하기 쉬운 인터페이스

## 설치 및 실행

### 1. 의존성 설치
```bash
pip install -e .
```

### 2. 애플리케이션 실행
```bash
python main.py
```

## 사용법

### 마우스 녹화
1. **녹화 시작**: "녹화 시작 (F1)" 버튼 클릭 또는 F1 키 누르기
2. **마우스 클릭**: 녹화하고 싶은 위치에서 마우스 클릭
3. **녹화 중지**: "녹화 중지 (F1)" 버튼 클릭 또는 F1 키 다시 누르기
4. **녹화 초기화**: "녹화 초기화" 버튼으로 녹화된 동작 삭제

### 자동 재생
1. **단일 재생**: "재생 시작 (F2)" 버튼 클릭 또는 F2 키 누르기
2. **자동 실행**: "자동 실행 시작" 버튼으로 반복 실행 모드 시작
3. **실행 간격 설정**: 설정 섹션에서 1-60분 사이의 간격 설정
4. **중지**: ESC 키로 모든 작업 즉시 중지

### 파일 관리
- **저장**: "저장" 버튼으로 녹화된 동작을 JSON 파일로 저장
- **불러오기**: "불러오기" 버튼으로 이전에 저장한 동작 파일 로드

### 전역 핫키
- **F1**: 녹화 시작/중지
- **F2**: 재생 시작/중지  
- **ESC**: 모든 작업 중지 (긴급 정지)

## 화면 구성

### 녹화 제어
- 녹화 시작/중지 버튼
- 녹화 초기화 버튼
- 현재 녹화 상태 표시

### 녹화된 동작
- 녹화된 마우스 클릭 동작들의 상세 정보
- 좌표, 버튼 종류, 시간 간격 표시
- 스크롤 가능한 텍스트 영역

### 재생 제어
- 단일 재생 버튼
- 자동 실행 버튼
- 현재 재생 상태 표시

### 설정
- 자동 실행 간격 설정 (1-60분)

### 파일 관리
- 저장, 불러오기, 종료 버튼

## 기술적 특징

- **pynput 라이브러리**: 마우스/키보드 이벤트 처리
- **tkinter GUI**: 직관적인 사용자 인터페이스
- **멀티스레딩**: 녹화/재생 중에도 GUI 반응성 유지
- **JSON 저장**: 녹화 데이터의 구조화된 저장
- **전역 핫키**: 다른 애플리케이션 사용 중에도 제어 가능

## 주의사항

- 관리자 권한이 필요할 수 있습니다 (특히 다른 애플리케이션 제어 시)
- 자동 실행 중에는 컴퓨터를 다른 용도로 사용하지 마세요
- ESC 키로 언제든지 모든 작업을 중지할 수 있습니다
- 녹화된 동작은 화면 해상도에 의존하므로 다른 해상도에서는 정확하지 않을 수 있습니다
